# Rating View in XML
```Kotlin
    <com.example.customviewtask.custom_view.rating.RatingView
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        app:numStars="5"
        app:rating="0"
        app:starSize="32dp"
        app:starSpacing="10dp"
        app:layout_constraintTop_toBottomOf="@id/txtSmth" />

```
